# methods-in-ecology-and-evolution
Command-line (Bash) and R scripts used for processing Oxford Nanopore long-read data from raw basecalled files through mapping, Medaka consensus polishing, SNP filtering, QC tier evaluation, and haplogroup/sex assignment analyses in vertebrate eDNA samples.

The command-line workflow (MEE_sequence_analysis_bash.txt) can be executed using the barcode FASTA files provided at the Figshare repository provided in the reviewer manuscript (nanopore_reads_October_2025.zip)

Downstream analyses and figure generation are implemented in the R Markdown script MEE_sequence_analysis.Rmd. This script can be run either using the outputs generated by the Bash pipeline and from the datasets available on Figshare:
eDNA_tissue_sexing_data.csv, nanopore_barcode_metadata.csv, and paired_samples_subset.csv
